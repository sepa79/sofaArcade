🏓 Arcade Pong 2P — ustawienia

Sterowanie

Gamepad: lewy analog / d-pad góra-dół • A = serw, a w grze: przytrzymaj A żeby naładować “smash”.

Klawiatura: P1: W/S + F (smash), P2: ↑/↓ + L (smash), serw/start: Spacja

Invaders Gun: pad B (button 1) • klawiatura: P1 E, P2 O

Tip: Gamepad API czasem wymaga kliknięcia w okno, żeby “złapać” focus.

Mecz

Jeśli włączone: trzeba wygrać przewagą 2 punktów.
Każde odbicie paletką zwiększa prędkość o (do max speed).

Tryb gry

Online P2P (bez serwera)

Wskazówka: uruchom przez http:// (np. python -m http.server), nie jako plik file://.
Status: offline
1) Host: „Utwórz Offer”, skopiuj → wklej po stronie Join.
2) Join: wklej Offer → „Utwórz Answer” → skopiuj do Host.
3) Host: wklej Answer → „Zastosuj Answer”.

Tryb Skippiego Wspaniałego

Skippy komentuje punktację i okazjonalnie dogryza. Klawisz: K (toggle).

Smash / Charge

Smash działa tylko, gdy piłka leci w stronę paletki i w momencie odbicia masz wciśnięte (ładowane) A/F/L.

Juice / feedback

Jeśli dźwięk nie działa: kliknij gdzieś na stronie (autoplay policy).

Start

Wciśnij Spację albo A, albo kliknij “Start”.

Podczas gry: Esc wraca do tego menu.
